This Plugin adds the ability to change the columns on the admin Dashboard screen. 

== Description ==
This Plugin adds the ability to change the vertical screen columns on the admin dashboard. To make a change to the amount of columns displayed click the screen options drop down and select the number of columns under screen layout.  

== Installation ==
Move to Plugin directory of the site and activate.
